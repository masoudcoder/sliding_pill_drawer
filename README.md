# Sliding Pill Drawer

A highly interactive and fluid side drawer package for Flutter. 

`sliding_pill_drawer` introduces a unique, draggable "pill-shaped" handle that sits seamlessly on the edge of your screen. Whether you need a sticky side-menu or a drawer that scrolls along with your content, this package handles the complex math and animations for you.

Perfect for educational apps, quiz screens, side-menus, and custom dashboards.

### ✨ Key Features
* **1:1 Drag Tracking:** The drawer perfectly follows the user's finger during horizontal swipes for a native, satisfying feel.
* **Sticky & Scrolling Modes:** Attach the pill handle to a specific item in your list, or make it stick to a fixed position on the screen.
* **Auto RTL/LTR Support:** Works flawlessly out of the box for both Left-to-Right (e.g., English) and Right-to-Left (e.g., Arabic, Persian) languages.
* **Highly Customizable:** Pass your own custom UI for both the drawer body and the pill handle, or use the beautiful default morphing animation.
